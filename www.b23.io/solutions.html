B23 Capabilities and Solutions

At the core of B23's capabilities is the ability to develop software. With Big Data, its not enough to just configure software, we think that's the easy part. B23 realizes that in order to get the full potential of your analytical tools, leveraging programming languages like Java, JavaScript, and Node is critical.

Integrating Big Data and Cloud solutions is very different than just configuring those solutions. That's where B23 is different.

Hadoop Ecosystem Expertise

B23 is focused on enabling the ever-growing and changing Hadoop ecosystem of platforms and tools. We believe Hadoop and extensions like YARN are inextricable linked to Big Data. We can help with the following:

  • Enable Cloud or on-premise Hadoop solutions
  • Enable query tools such as Hive, Pig, and Impala
  • Develop custom MapReduce algorithms
  • Develop custom User Defined Functions (UDF's)

Data Streaming and Messaging Expertise

New techniques for ingesting data are allowing organizations to analyze data in near real-time. B23 is focused on emerging data streaming, messaging, and real-time analytical frameworks including:

  • Apache Storm
  • Apache Samza
  • RabbitMQ

Scalable Data Store Expertise

B23 is constantly evaluating new data store technologies that can provide competitive advantages to customers based on their specific needs. We have experience with:

  • MongoDB, GridFS, and Aggregation Framework
  • Apache Accumulo
  • Apache Hbase
  • Neo4j

Cloud Enablement Expertise

B23 can help enable Cloud hosted environments to support Big Data applications. This includes infrastructure and platform service enablement. We can help:

  • Secure and enable private Cloud enclaves
  • Automate deployment of applications within Cloud Platform services
  • Manage, monitor, and audit Cloud environments

DevOps and Infrastructure Automation Expertise

Without automation, the economic benefits for elasticity go out the window. B23 is focused on automating the underlying infrastructure to help loosely coupled applications scale horizontally efficiently and cost effectively. We have expertise in:

  • Puppet Labs
  • Salt Open Source
  • Nagios

Micro Application Development

B23 knows there are times when it's required to develop an inexpensive, functional application to quickly analyze collected data. Examples include:

  • Collecting log data to analyze for security and compliance purposes
  • Credential verification
  • Automated infrastructure asset tagging
  • Developing a custom web interface to view the contents of a data store

We support agile development of these applications so our customers can take the first step to understanding their Big Data.
